## About

AI Antonio's Intelligence is an interactive music production and spiritual coaching platform featuring:
- Real-time video streaming
- Pisces-guided rhyme coaching
- Beat marketplace
- Zodiac-based creative development
- Firebase real-time chat with Antonio's AI twin

## Tech Stack

- **Frontend**: Vanilla HTML, CSS (Tailwind), JavaScript
- **Backend**: Firebase (Firestore, Authentication)
- **APIs**: Google Gemini AI, Cloud TTS, Imagen
- **Media**: Video.js for beat playback
- **Recording**: MediaRecorder API

## Getting Started

### Prerequisites

- A web browser (Chrome, Edge, or Firefox recommended)
- VSCode with Live Server extension
- Firebase configuration (provided via runtime)

### Running the Application

1. Open the project in VSCode
2. Right-click on `index.html`
3. Select "Open with Live Server"
4. The app will open at `http://localhost:5501/index.html`
5. Grant camera and microphone permissions when prompted
